# Noshahi Sound System Website

This is a responsive, mobile-first website for Noshahi Sound System, built with HTML, CSS, and Bootstrap 5.

## How to Run
Simply double-click `index.html` to open it in your web browser. No server installation is required.

## Customization
- **Images**: Replace the placeholder images in `index.html` and `style.css` with your own photos.
- **Colors**: Edit the variables in `style.css` (e.g., `--primary-color`) to change the color scheme.
- **Contact Form**: The form is currently a frontend template. To make it functional, you can use a service like [Formspree](https://formspree.io/) or integrate with a backend.
